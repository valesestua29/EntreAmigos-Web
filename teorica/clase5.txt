---------------- 5. Medidas, Colores, Fondos, Fuentes e Íconos -----------------

Propiedades basicas -----

    color: permite cambiar el color a los textos de los elementos html

            color: red ;

    text-decoration: se usa para establecer el formato de textos

            underline: subrayado
            overline: suprarrayado
            line-through: tachado
            blick: parpadeo

    list-style: permite modificar la viñeta de las listas

            none ; circle; georgian ;

    text-aling: permite alinear el textos

            center; justify; right; left; 


Unidades de medida -----

    unidades absolutas: son fijas y no cambian segun el tamaño del viewport (px, cm, mm, in, pt, pc)
    unidades relativas: dependen del entorno y son mas flexibles adaptandose al tamaño de pantalla (em, rem, %)
    unidades flexibles: relativas al tamaño de la ventana del navegador (vw, vh)


 Fondos -----

    color de fondo: para aplicar un color solido al fondo del elemento
        background-color : #valor ;

    imagen de fondo: para aplicar una imagen de fondo
        background-image: url ('ruta');


Fuentes y tipografias -----

    La tipografia es uno de los pilares del diseño web

    font-size: define el tamaño del texto
    font-style: define el estilo (normal, italic, oblique)
    font-family: especifica la familia tipografica
    font-weight: controla el grosor del texto (100 a 900)


    incorporar tipografias

        visitar google fonts
        copiar el enlace proporcionado e incluirlo en <head> de html
        aplicar usando font-family


Iconos -----

    Permiten mejorar la usabilidad y el diseño de tu sitio. Se pueden integrar fácilmente con bibliotecas como 
    Font Awesome o Flaticon.

    Font Awesome: Solo necesitas agregar un enlace en tu HTML
        <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0-beta3/css/all.min.css">
        <i class="fas fa-shopping-cart"></i>


    Flaticon: permite descargar íconos y agregarlos como imágenes.
        <img src="https://cdn-icons-png.flaticon.com/512/1170/1170627.png" alt="Ícono de libro" width="50">
